Wyam is a simple to use, highly modular, and extremely configurable static content generator. It can be used to generate web sites, scaffold development projects, create ebooks, and much more.

It's currently under heavy development, but is usable today. Read on for more information if you want to get started now and help influence it's future direction.

Features

  • Flexible script-based configuration
  • Lots of modules for things like reading and writing files, handling frontmatter, and manipulating metadata
  • Integrated YAML parser
  • Support for multiple templating languages including Razor
  • Integrated web server for previewing output
  • Integrated file watching and regeneration
  • Full NuGet support
  • Easily extensible
  • Embeddable engine
